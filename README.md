
# Quiz App

That is time controlled quiz application.


## Run Locally

Clone the project

```bash
  git clone https://github.com/Serifeyaman/quiz-app
```

Go to the project directory

```bash
  cd quiz-app
```

Install dependencies

```bash
  npm install
```
or

```bash
  yarn install
```

Start the server

```bash
  npm run start
```
or

```bash
  yarn start
```


## Tech Stack

**Client:** React, TailwindCSS
